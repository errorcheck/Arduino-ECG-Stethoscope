# Arduino-ECG-Stethoscope
> "Arduino como Estetoscopio Digital: ECG, BPM y Arritmia Solo con Potenciómetro"  
> Real-time heartbeat monitoring with LED sync and Serial Plotter visualization.
> 
**Turn a potentiometer into a digital stethoscope**   Real-time ECG simulation + BPM calculation + beating LED
